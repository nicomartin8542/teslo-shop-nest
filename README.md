# Teslo API

1. Clonar proyecto
2. `npm install `
3. Clonar el archivo `.env.template` por `.env` y agregar el valor a cada una de la variables
4. Levantar la base de datos

```bash
  docker-compose up -d
```

5. Levantar app con `npm run start:dev`
